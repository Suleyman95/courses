Создать файл settings.php
Этот файл будет принимать следующие переменные по методу POST:
email
password
phone
about
note

Используя подходящие функции необходимо изменить поля в базе данных на те, что пришли в этих переменных.

Чтобы выполнить изменения пользователь должен быть авторизован (is_auth).

Чтобы узнать для какого пользователя делаются изменения можно извлечь student_id из сессии (добавляется туда в момент авторизации в authorize())
